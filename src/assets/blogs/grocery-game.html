🛒👩‍🍳 “Recipe Rush: Online Grocery Ordering Game”

🎯 Game Objective

You play as a home chef who must:

  • Select a recipe,

  • Order the correct grocery items from an online grocery store,

  • Stay within your budget,

  • And complete the recipe before time runs out!

It's a fun text-based simulation game teaching decision-making, budgeting, and basic input/output in C++ — great for beginners.

🧱 Features

  • Multiple recipes (e.g., Spaghetti, Chicken Curry, Vegetable Salad)

  • Each recipe has a list of ingredients.

  • You’re given a budget and time.

  • You can browse a grocery store catalog.

  • Buy items wisely. Buying wrong items or overspending results in failure.

  • Win if you collect all items for the recipe on time and under budget!

🧮 C++ Concepts Used

  • Structs

  • Vectors

  • Loops

  • Conditionals

  • Functions

  • Basic randomization

  • Simple gameplay mechanics

🧾 Sample Code: recipe_rush.cpp

#include <iostream>
#include <vector>
#include <string>
#include <iomanip>
using namespace std;

struct Item {
    string name;
    double price;
};

struct Recipe {
    string name;
    vector<string> ingredients;
    double maxBudget;
};

// Store items
vector<Item> store = {
    {"Tomato", 1.5}, {"Onion", 1.0}, {"Chicken", 5.0}, {"Salt", 0.5},
    {"Pepper", 0.7}, {"Spaghetti", 2.5}, {"Garlic", 1.2}, {"Oil", 2.0},
    {"Rice", 3.0}, {"Carrot", 1.3}, {"Lettuce", 2.2}, {"Milk", 1.8}
};

// Available recipes
vector<Recipe> recipes = {
    {"Spaghetti", {"Spaghetti", "Tomato", "Onion", "Garlic", "Salt"}, 10.0},
    {"Chicken Curry", {"Chicken", "Onion", "Garlic", "Salt", "Pepper", "Oil"}, 12.0},
    {"Veggie Salad", {"Lettuce", "Tomato", "Carrot", "Salt", "Oil"}, 8.0}
};

void displayStore() {
    cout << "\n🛒 Grocery Store Catalog:\n";
    cout << setw(15) << left << "Item" << "Price\n";
    cout << "-------------------------\n";
    for (size_t i = 0; i < store.size(); ++i) {
        cout << i + 1 << ". " << setw(12) << left << store[i].name 
             << "$" << fixed << setprecision(2) << store[i].price << endl;
    }
}

void displayRecipe(const Recipe& recipe) {
    cout << "\n📜 Recipe Selected: " << recipe.name << endl;
    cout << "Ingredients needed:\n";
    for (const string& ing : recipe.ingredients)
        cout << "- " << ing << endl;
    cout << "💵 Budget: $" << recipe.maxBudget << endl;
}

bool allIngredientsCollected(const vector<string>& needed, const vector<string>& bought) {
    for (const string& item : needed) {
        if (find(bought.begin(), bought.end(), item) == bought.end())
            return false;
    }
    return true;
}

int main() {
    cout << "🎮 Welcome to 'Recipe Rush: Online Grocery Game'!\n";

    // Step 1: Choose Recipe
    cout << "\nChoose a recipe:\n";
    for (size_t i = 0; i < recipes.size(); ++i) {
        cout << i + 1 << ". " << recipes[i].name << endl;
    }
    int choice;
    cin >> choice;
    if (choice < 1 || choice > recipes.size()) {
        cout << "❌ Invalid choice. Exiting...\n";
        return 0;
    }

    Recipe selected = recipes[choice - 1];
    vector<string> cart;
    double total = 0.0;

    displayRecipe(selected);

    int turns = 7; // limited shopping attempts
    while (turns-- > 0 && !allIngredientsCollected(selected.ingredients, cart)) {
        displayStore();
        cout << "\n🧺 Your current cart: ";
        for (const string& item : cart) cout << item << ", ";
        cout << "\n💸 Total spent: $" << total << " | 🕒 Turns left: " << turns + 1 << endl;

        cout << "\nEnter item number to add to cart (0 to skip): ";
        int itemChoice;
        cin >> itemChoice;

        if (itemChoice == 0) continue;
        if (itemChoice < 1 || itemChoice > store.size()) {
            cout << "❌ Invalid item!\n";
            continue;
        }

        Item selectedItem = store[itemChoice - 1];

        // Check budget
        if (total + selectedItem.price > selected.maxBudget) {
            cout << "⚠️ You can't afford this item. Try something cheaper.\n";
            continue;
        }

        // Add item
        cart.push_back(selectedItem.name);
        total += selectedItem.price;
        cout << "✅ " << selectedItem.name << " added to cart!\n";
    }

    // Game result
    if (allIngredientsCollected(selected.ingredients, cart)) {
        cout << "\n🎉 Congratulations! You successfully ordered all items for " << selected.name << " within budget.\n";
    } else {
        cout << "\n❌ You failed to collect all ingredients. Better luck next time!\n";
    }

    return 0;
}

🧠 How the Game Works

👨‍🍳 1. Recipe Selection

The player chooses from 3 recipes, each with specific ingredients and a budget.

🛍️ 2. Grocery Store Catalog

A list of available groceries with prices is shown. Items are selected via number input.

🧮 3. Budget Tracking

The player has to shop carefully. Exceeding the budget prevents item purchase.

⏲️ 4. Time Pressure (Turns)

The player has limited turns to complete shopping (simulating delivery cut-off).

🏆 5. Win/Lose

If all recipe ingredients are collected within budget and turns, the player wins!
